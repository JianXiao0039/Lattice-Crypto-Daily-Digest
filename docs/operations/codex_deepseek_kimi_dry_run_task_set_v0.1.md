# Codex / DeepSeek-Claude / Kimi Dry Run Task Set v0.1

Every operator must use the same command sequence unless a command is unavailable. Use PowerShell in this repository unless explicitly translating to CMD or Git Bash.

## Task 0 - Boundary Confirmation

Confirm:

- working directory is `D:\Code\CodexProjects\lattice-crypto-daily-digest`;
- no private paths will be read or written;
- no `git add`, `git commit`, `git push`, or `git tag`;
- no background automation;
- no ranking, source fetcher, taxonomy, query expansion, or negative keyword changes.

## Task 1 - Pre-run Checks

```powershell
Set-Location "D:\Code\CodexProjects\lattice-crypto-daily-digest"
git status -sb
python --version
python -m lattice_digest.workflow doctor
python -m lattice_digest.workflow status
```

If `workflow status` is unavailable, report it and continue only if `doctor` passes.

## Task 2 - Daily Latest Safe Run

```powershell
python -m lattice_digest.run --since 36h --output markdown,json --send none
```

Expected artifacts:

- `data/YYYY-MM-DD.json`
- `digests/YYYY-MM-DD.md`
- `papers.db`
- source-health audit if generated by the workflow

Source failures must be reported as degraded or source-starved; never interpret a failed source as proof of no relevant papers.

## Task 3 - Weekly Dry-run

```powershell
python -m lattice_digest.workflow weekly --low-load --skip-hygiene
```

Do not execute weekly unless the phase explicitly authorizes it.
