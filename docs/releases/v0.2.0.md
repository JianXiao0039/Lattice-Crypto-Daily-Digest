# v0.2.0 - Research Library Interoperability Stable Release

## 中文说明

## 1. 发布定位

v0.2.0 是 stable release，不是 release candidate。它把 `lattice-crypto-daily-digest` 从“能稳定生成格密码科研日报”的工具，推进为“能把日报数据沉淀到研究文献库生态”的互操作层。

本版本稳定的是文件式文献库导出、深度格密码 taxonomy、Library Export Quality Audit、Zotero compatibility layer 和 Zotero Manual Import QA。它仍然是研究 triage 与 workflow 自动化工具，不声称能自动保证元数据完全正确，也不声称能验证密码学安全结论。

## 2. 从 v0.1.0 到 v0.2.0 的变化

v0.1.0 重点解决公开部署、自动化日报、本地运行、GitHub Actions provisional 日报、本地 authoritative backfill 和基础运行稳定。

v0.2.0 重点解决文献库导出与互操作：

- Zotero / Obsidian / BibTeX / RIS / CSL JSON 文件式互操作。
- 深度格密码 taxonomy 与稳定 `library item` schema。
- Library Export Quality Audit 与 Zotero Manual Import QA。
- 从每日 digest records 到长期文献库、Obsidian、Idea Bank、Paper Plan 的研究资料沉淀。

## 3. 核心能力

- Daily digest：中文格密码论文日报。
- Local authoritative backfill：本地权威回填 GitHub provisional 日报。
- Weekly brief：周报研究简报工作流。
- Obsidian paper cards：Obsidian 论文卡片工作流。
- Idea Bank：长期研究想法沉淀。
- Paper Plan：把 idea 升级为论文计划草案。
- Research artifact scaffold：从 Paper Plan 生成研究项目骨架。
- Stable Library Export Layer：稳定文献库导出层。
- Zotero compatibility layer：离线 Zotero 兼容导出层。
- Library Export Quality Audit：字段质量、标签误报和导入风险审计。
- Zotero Manual Import QA：文件式手动导入 Zotero 的验收流程。

## 4. 支持格式

- Markdown。
- JSON。
- `library-items.json`。
- CSL JSON。
- BibTeX。
- RIS。
- Zotero-style JSON。
- `zotero-tags.json`。
- `import-report.md`。
- audit reports。

## 5. 格密码标签体系

v0.2.0 的 deterministic taxonomy 覆盖：

- Lattice Problems：LWE、RLWE、MLWE、SIS、Module-SIS、Ring-SIS、NTRU、SVP、CVP、BDD、q-ary lattice、module lattice。
- Lattice Reduction：BKZ、LLL、G6K、fplll、sieving、enumeration、pruning、GSA、lattice estimator。
- Lattice Cryptanalysis：primal attack、dual attack、hybrid attack、secret recovery、distinguisher、sparse LWE、sample selection。
- PQC Schemes：ML-KEM、Kyber、ML-DSA、Dilithium、Falcon、FN-DSA、FrodoKEM、Saber、NIST PQC。
- Lattice Primitives：commitment、Module-SIS commitment、chameleon hash、FHE、CKKS、BFV、BGV、TFHE、lattice-based ZK、ZK-friendly PQ privacy primitives。
- Implementation Security：constant-time、side-channel、fault attack、masking、NTT、production audit、benchmark、artifact evaluation。
- AI4Lattice：Transformer LWE、Swin-guided coordinate selection、negative-cyclic modeling、hybrid ranking、neural lattice reduction、learned pruning、BKZ parameter prediction。
- Research Workflow Tags：Must Read、Recommended Read、Skim、Archive、Paper Idea、Experiment Candidate、TODO_VERIFY。

## 7. Zotero 当前支持方式

v0.2.0 当前支持的是 file-based manual import。

明确边界：

- 当前是 file-based manual import。
- 当前不是 Zotero XPI plugin。
- 当前不自动调用 Zotero Web API。
- 推荐先用 CSL JSON / BibTeX / RIS 手动导入。
- 导入后人工检查 title、authors、year、DOI、URL、abstract、tags、notes。

`zotero_items.json` 是面向未来 Zotero Web API 的离线 editable-style JSON，不代表当前已经支持 Zotero API push。

## 8. 不包含的功能

- Zotero XPI plugin。
- Zotero Web API push。
- automatic PDF attachment import。
- browser extension。
- cloud-hosted dashboard。
- security proof verification。

## English Summary

v0.2.0 is the stable Research Library Interoperability release of Lattice Crypto Daily Digest. Compared with v0.1.0, which focused on deployment and daily digest automation, v0.2.0 stabilizes the file-based library export layer, deterministic lattice-cryptography taxonomy, CSL JSON / BibTeX / RIS / Zotero-style JSON export, library export audit, and Zotero manual import QA.

The supported Zotero workflow is file-based manual import: export CSL JSON, BibTeX, or RIS, import into Zotero, and manually verify title, authors, year, DOI, URL, abstract, tags, and notes. v0.2.0 does not include Zotero Web API sync, Zotero XPI plugin, automatic PDF attachment import, browser extensions, cloud dashboards, or cryptographic security-proof verification.

The release is intended for lattice cryptography researchers, PQC students, AI4Lattice researchers, PhD applicants building literature workflows, and anyone tracking LWE/RLWE/MLWE/SIS/BKZ/PQC implementation papers.
