## What is stable in v0.1.0

- Daily lattice crypto digest：近 24-36 小时格密码论文抓取、过滤、去重、排序和中文日报输出。
- Source Health：red/yellow/green 数据源诊断。
- Local authoritative backfill：本地 Windows PowerShell 权威回填。
- GitHub Actions provisional digest：云端保底日报。
- Backfill audit：比较 GitHub provisional 和本地 authoritative backfill 的差异。
- Idea Bank：从 digest 记录沉淀研究想法。
- Paper Plan：把 idea 升级为保守研究计划。
- Research Artifact scaffold：把 Paper Plan 转换为可复现实验和论文写作骨架。
- Public Deployment Guide：公开部署、配置、排错和安全说明。

## What is experimental

- Weekly Research Brief：作为研究工作流方向保留，具体命令以当前分支实际脚本为准。
- Obsidian paper card export：作为研究工作流方向保留，具体命令以当前分支实际脚本为准。
- Idea Bank、Paper Plan 和 Research Artifact 的启发式生成仍是研究规划辅助，不是论文结论。
- Source coverage depends on external APIs and local/GitHub network state.

## Recommended deployment modes

### Local manual run

```powershell
Set-Location "D:\Code\CodexProjects\lattice-crypto-daily-digest"
python -m pytest tests --basetemp=.pytest_tmp
python -m lattice_digest.run --since 36h --output markdown,json --send none
```

### Local scheduled run

可以在本机用 Windows PowerShell 或外部任务工具定期运行日报命令。项目不要求 Codex 自动化模块作为运行依赖。

### GitHub Actions provisional run

GitHub Actions 适合作为保底 collector。它生成 `quality_status=provisional` 的日报，适合在本地电脑不开机时维持连续记录。

### Local authoritative backfill

```powershell
Set-Location "D:\Code\CodexProjects\lattice-crypto-daily-digest"
powershell.exe -ExecutionPolicy Bypass -File scripts\run_local_digest_backfill.ps1 -Days 5
```

本地 authoritative backfill 通常比 GitHub runner 覆盖更完整，适合用来替换或增强 provisional 日报。

## API recommendations

- IACR ePrint: no API key required, core source.
- arXiv: no key, but rate/backoff needed.
- Crossref: optional email/user-agent if supported by future code paths.
- Semantic Scholar: recommended API key via `SEMANTIC_SCHOLAR_API_KEY`.
- OpenAlex: avoid paid-only sort; use free-compatible queries.
- DBLP: useful but may be sparse for daily windows.

- Semantic Scholar works best with API key.
- This tool is research triage, not a formal bibliographic database.
