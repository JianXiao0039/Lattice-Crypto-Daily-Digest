# v0.4 CI Green Requirements v0.1

## Required Jobs

- Ubuntu latest with Python 3.11: green.
- Windows latest with Python 3.11: green.

## Required Steps

1. checkout succeeds;
2. package and `.[dev]` installation succeeds;
3. `python -m pytest tests --basetemp=.pytest_tmp` succeeds;
4. release hygiene succeeds;
5. `git diff --check` succeeds.

## Test Scope

- retain `--date` CLI and exact-date tests;
- retain source-starved artifact tests;
- retain reliability dashboard tests;
- retain Git tracking and release-candidate tests;
- do not skip Linux-only failures or remove matrix coverage.

## Release Gate

CI must be green for the exact commit selected for release, not merely for an earlier commit.
