# Academic Research Suite Public Repository Usage Policy v0.1

## Allowed Phase 13A Use

- experiment-agent reproducibility planning;
- evidence-schema review;
- academic-paper-reviewer claim and overstatement review.

## Disabled

- `ARS_CODEX_FULL_RUNTIME`;
- `ARS_CODEX_AGENT_TEAM`;
- `ARS_CODEX_HOOKS`;
- external cross-model review;
- automatic release-engineering edits by ARS;
- vendoring the skill into this repository.

## Authority Boundary

ARS provides structured review guidance only. Repository tests, release hygiene, Git history, GitHub Actions metadata, and retained artifacts are authoritative for engineering decisions.
