# v0.5 Weekly Rationale Upgrade Plan v0.1

Status: controlled future integration plan.

Weekly synthesis impact:

- Reuse rationale generated from daily records when present.
- Do not reinterpret source-starved days as "no relevant papers".
- Use rationale to improve weekly "why this matters" and reading-queue handoff summaries.
- Preserve weekly coverage, source-health, and handoff packet semantics.

Weekly output should distinguish:

- abstract-supported weekly recommendation,
- conclusion-supported weekly recommendation,
- metadata-only or title-only carryover,
- peripheral FHE/application papers,
- TODO_VERIFY items requiring paper reading.

Gate:

Weekly integration should follow Daily integration, not precede it. Weekly synthesis should not invent missing rationale for records that lack abstract/conclusion evidence.
