# v0.6 Top-Paper Bilingual Rendering Policy v0.1

Status: `top_paper_bilingual_rendering_ready`.

Monthly core papers render full bilingual blocks for the first five core papers and all A-class core papers.

Weekly top A-level papers render the same bilingual block.

The required block shape is:

中文：

* 论文大致工作：
* 核心创新点：
* 与本雷达关系：
* 建议：
* TODO_VERIFY：

English:

* Paper work summary:
* Core novelty:
* Radar relevance:
* Recommendation:
* TODO_VERIFY:

Daily and lower-priority B/C records may continue using compact rationale unless a future phase explicitly broadens rendering.
