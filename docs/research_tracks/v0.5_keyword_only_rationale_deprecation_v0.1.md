# v0.5 Keyword-Only Rationale Deprecation v0.1

Status: proposal.

Deprecated pattern:

> Matched keywords: LWE, lattice, signature.

Reason:

Keyword hits are useful evidence, but they do not explain the paper's problem, method, contribution, or reading value. They also risk over-promoting weak FHE applications, generic ML papers, or generic signature papers.

Replacement rule:

- Keep keyword hits as supporting evidence.
- Add evidence-basis labels.
- Use abstract/conclusion text when available.
- If only title/metadata exists, say so explicitly.
- Add TODO_VERIFY when full paper text is unavailable.

Compatibility:

Existing keyword fields remain available for ranking explainability and audit trails. This proposal changes explanation quality, not scoring semantics.
