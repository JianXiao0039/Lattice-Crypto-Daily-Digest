# v0.5 Manual Low-Load Runbook v0.1

Status: `compliant_manual_retry_policy_ready`.

This runbook is manual-only. It does not create scheduled automation or background retries.

## Commands

```powershell
python scripts\probe_source_health.py --low-load
python scripts\verify_durable_artifacts.py --date 2026-06-15 --week 2026-W25 --month 2026-06
```

## Interpretation

- `rate_limited`: wait; honor `Retry-After`; reduce load.
- `ssl_failure`: inspect CA/proxy; do not disable SSL verification by default.
- `auth_failed` or `missing_key`: enrichment unavailable; do not mark papers irrelevant.
- `empty_response`: verify query/date filters.
- `parser_failure`: inspect parser and response format.
- `source-starved`: treat output as incomplete evidence and add `TODO_VERIFY`.

## Anti-Abuse Guardrails

- No proxy rotation.
- No fake User-Agent rotation.
- No CAPTCHA or access-control bypass.
- No hidden browser automation.
- No aggressive repeated retries.
