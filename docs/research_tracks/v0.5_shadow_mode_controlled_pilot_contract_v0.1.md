# v0.5 Shadow-Mode Controlled Pilot Contract v0.1

## Purpose

Run experimental track rules manually against retained repository evidence without changing the paper-radar production pipeline.

## Contract

- Invocation is explicit and manual-only.
- Default input is the self-contained Phase 13E prediction snapshot.
- Default rules are `experiments/v0_5_shadow_track_rules_v0.2.json`.
- Default output is `audits/shadow/v0_5_controlled_pilot/`.
- Output may also be directed to an explicit path outside the repository or under `docs/research_tracks/`.
- Output under `data/`, `digests/`, `handoffs/`, `src/`, or the package bridge is rejected.
- The script does not import production classification, ranking, source, workflow, or ARS runtime code.
- Production labels are read-only comparison evidence from the retained Phase 13E snapshot.
- Human-gold metrics are forbidden unless explicit `user_confirmed` or `user_corrected` labels exist.

## Non-Claims

Agreement is not accuracy. Codex-reviewed or production labels are not human gold. Completion does not authorize production promotion.
