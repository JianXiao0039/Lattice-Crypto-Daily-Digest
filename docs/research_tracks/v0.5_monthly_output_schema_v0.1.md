# v0.5 Monthly Output Schema v0.1

JSON output:

```json
{
  "schema_version": 1,
  "month": "YYYY-MM",
  "generated_at": "...",
  "input_daily_files": [],
  "input_weekly_files": [],
  "missing_days": [],
  "total_unique_records": 0,
  "class_counts": {},
  "direction_counts": {},
  "source_health_summary": {},
  "core_papers": [],
  "reading_priority": {},
  "trend_summary": [],
  "rationale_policy": {},
  "TODO_VERIFY": {}
}
```

Markdown output sections:

- Executive Summary
- Core Papers of the Month
- Direction Trends
- Reading Priority
- Source Health Summary
- TODO_VERIFY

Compatibility:

Monthly JSON is a new artifact type under `data/monthly/`. Daily and Weekly JSON schemas are unchanged.
