# v0.4 GitHub Actions Failure Triage v0.1

## Failure

Ubuntu/Python 3.11 failed one test because a reliability-baseline assertion required Windows backslashes.

## Root Cause

Repository-relative artifact paths were serialized using the host operating system separator. The test then encoded the Windows representation as the only accepted value.

## Fix

- serialize repository-relative artifact paths with `Path.as_posix()`;
- assert POSIX-form paths on all platforms;
- retain Ubuntu and Windows CI coverage.

## Verification

- Linux/Python 3.11: 446 passed after the fix;
- Windows focused tests: passed;
- full Windows suite and GitHub Actions: required before tag readiness.

## Workflow Decision

No workflow change. The existing matrix correctly exposed a portability defect and should remain intact.
