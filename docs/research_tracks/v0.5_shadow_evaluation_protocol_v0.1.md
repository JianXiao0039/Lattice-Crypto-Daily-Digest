# v0.5 Shadow Evaluation Protocol v0.1

## Inputs

- Repository-grounded Phase 13B sample.
- Experimental rule file.
- Current production classifier output for comparison only.

## Outputs

- Shadow prediction and explanation per record.
- Production-shadow disagreement categories.
- Review-priority queue.

## Valid Metrics

Before human review, report only agreement/disagreement counts, coverage, and review priority. Precision, recall, F1, and accuracy require `user_confirmed` or `user_corrected` labels.

## Reproducibility

Run:

```powershell
python scripts\run_v0_5_shadow_track_classifier.py
python scripts\build_v0_5_human_annotation_pack.py
```

The experiment is deterministic for fixed sample and rules. No network is required.
