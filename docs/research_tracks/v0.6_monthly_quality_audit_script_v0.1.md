# v0.6 Monthly Quality Audit Script v0.1

## Script

`scripts/audit_monthly_rationale_quality.py`

## Manual Commands

```powershell
python scripts\audit_monthly_rationale_quality.py --latest
python scripts\audit_monthly_rationale_quality.py --month 2026-06
```

## Outputs

- `audits/monthly-quality/YYYY-MM.json`
- `audits/monthly-quality/YYYY-MM.md`
- `docs/reports/phase-13r-rationale-quality-scorecard.md`
- `docs/reports/phase-13r-real-paper-case-audit-log.json`
- `docs/research_tracks/v0.5_monthly_real_case_quality_audit_v0.1.md`

## Checks

- artifact availability and parseability;
- monthly Markdown section presence;
- keyword-only rationale;
- evidence basis;
- TODO_VERIFY;
- title-only and metadata-only conservatism;
- weak relevance overpromotion;
- conclusion claims without conclusion evidence;
- bilingual top-paper policy;
- source-health caveats;
- reading-action usefulness;
- absence of user annotation dependency.

## Runtime Boundaries

The script does not call the network, external LLMs, or private paths. It reads local monthly artifacts and writes local audit reports only.
