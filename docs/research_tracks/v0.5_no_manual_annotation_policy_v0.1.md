# v0.5 No Manual Annotation Policy v0.1

Status: `no_manual_annotation_dependency_confirmed`.

Phase 13M is a reading usability phase. It does not create or require manual labels, human-gold labels, classifier training data, or annotation metrics.

## Forbidden Fields

- `user_label`
- `human_gold_label`
- `user_confirmed`
- `user_corrected`
- `manual_annotation_status`

## Allowed Local Fields

- `reading_status`
- `review_status`
- `obsidian_note_path`
- `zotero_key`
- optional blank `user_notes` if a future local-only convention requires it.

These are workflow fields, not classifier labels.
