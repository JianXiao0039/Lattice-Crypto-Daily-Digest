# v0.5 Track Gold Sample Schema v0.1

| Field | Requirement |
|---|---|
| `paper_identifier` | Existing repository ID |
| `title` | Existing repository title |
| `source` | Existing source label |
| `candidate_tracks` | Plausible tracks before adjudication |
| `primary_track` | Best manual label |
| `secondary_track` | Optional secondary label |
| `relevant` | `yes`, `no`, or `ambiguous` |
| `positive_evidence` | Explicit lattice/PQC/attack evidence |
| `exclusion_evidence` | Generic/noise or missing-anchor evidence |
| `expected_handoff_destination` | Radar, quick-paper, background, or exclusion |
| `annotation_confidence` | `high`, `medium`, or `low` |
| `annotator_note` | Scope and caution note |
| `TODO_VERIFY` | Original-source verification still needed |

The schema is an evaluation artifact. It is not production taxonomy or ranking logic.
