# v0.6 Bilingual No-Overclaim Policy v0.1

Status: `no_overclaim_policy_enforced`.

The bilingual rationale layer must not overstate:

- novelty;
- proof status;
- security reductions;
- parameter claims;
- benchmark claims;
- implementation claims;
- conclusion-derived claims when no conclusion field is present.

If evidence is weak, render:

- 核心创新点：证据不足，暂不能确认；需要阅读正文验证。
- Core novelty: insufficient evidence from available metadata; verify in the full paper.

The helper keeps TODO_VERIFY visible even for abstract-supported top papers because abstract support is not the same as verified proof, parameter, benchmark, or implementation evidence.
