# Weekly Automation Observation Checklist v0.1

## Before Observation

- [ ] 使用当前 weekly artifact
- [ ] 不写入私有目录
- [ ] 不执行 git 提交类命令

## Weekly Observation Checks

- [ ] `data\weekly\YYYY-Wxx.json` 存在
- [ ] `handoffs\weekly\YYYY-Wxx-handoff-packets.json` 存在
- [ ] `handoffs\weekly\YYYY-Wxx-handoff-packets.md` 存在
- [ ] weekly handoff 候选数可见
- [ ] 若为空，缺失原因明确
- [ ] 若 weekly 输入 source-starved，输出明确标注
- [ ] 不把空 handoff 解释为“没有候选论文”

## Post-Observation Interpretation

- [ ] 周汇总是否放大了旧的 red 历史
- [ ] 当前周 handoff 是否仍可用
- [ ] 是否需要重新运行 handoff
