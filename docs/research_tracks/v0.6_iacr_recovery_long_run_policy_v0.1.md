# v0.6 IACR Recovery Long-Run Policy v0.1

Status: `iacr_recovery_policy_ready`.

IACR ePrint recovery must distinguish:

- RSS/latest unreachable;
- RSS/latest parser failure;
- failed-attempt guard;
- cache hit;
- cache freshness if available;
- zero latest records;
- failed/0.

`failed/0` is not evidence that IACR has no relevant papers.

When IACR is unavailable, the radar may continue with degraded evidence and explicit source-health caveats.
