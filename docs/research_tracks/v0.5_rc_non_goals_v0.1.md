# v0.5 RC Non-Goals v0.1

Phase 13N explicitly excludes:

- private PhD application workflows;
- Module-SIS artifact writing;
- manual annotation workflows;
- human-gold metrics workflows;
- shadow classifier productionization;
- source fetcher changes except previously documented narrow reliability fixes;
- ranking score changes;
- ranking threshold changes;
- taxonomy semantic changes;
- query expansion changes;
- negative keyword behavior changes;
- scheduled tasks, watchers, cron jobs, startup tasks, or background services;
- Supervisor-Skills or academic-research-skills-codex runtime dependencies;
- external LLM calls;
- tag creation or movement.
