# v0.5 Shadow-Mode Output Schema v0.1

## Bundle

- `manifest.json`: run identity, manual mode, rule/input provenance, counts, output list, and non-production flags.
- `predictions.json`: all controlled shadow predictions and retained comparison labels.
- `disagreements.json`: non-exact comparisons only.
- `summary.md`: human-readable counts and gate reminder.

## Record Fields

- record and sample identifiers;
- title and source;
- production primary track from retained evidence;
- shadow primary and secondary tracks;
- confidence and explanation;
- matched primary, support, and exclusion rules;
- disagreement category;
- human-review status and valid-gold boolean;
- source provenance and `TODO_VERIFY`.

No field changes a production record.
