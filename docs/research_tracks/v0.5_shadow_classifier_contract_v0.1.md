# v0.5 Shadow Classifier Contract v0.1

## Scope

The shadow classifier is an offline evaluator. It reads the Phase 13B sample and experimental rules, emits predictions and explanations, and compares them with current production behavior.

## Isolation

- Entry point: `scripts/run_v0_5_shadow_track_classifier.py`.
- Rules: `experiments/v0_5_shadow_track_rules.json`.
- Default output: `audits/shadow/`.
- No import or call from `src/lattice_digest`, Daily, Weekly, source fetchers, ranking, reading queues, or handoff generation.
- Disabled by default and manually invoked only.
- Academic Research Suite is not a runtime dependency.

## Human Boundary

Shadow predictions use `queued_for_user`. They never assign `user_confirmed` or `user_corrected`, and never count as human gold.
