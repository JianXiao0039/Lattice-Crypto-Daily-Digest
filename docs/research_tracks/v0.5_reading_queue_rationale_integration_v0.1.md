# v0.5 Reading Queue Rationale Integration v0.1

Status: `rationale_integrated`.

The reading queue imports deterministic rationale fields from `lattice_digest.recommendation_rationale`.

## Added Fields

- `rationale_problem`
- `rationale_method`
- `rationale_contribution`
- `rationale_relevance`
- `rationale_caveat`
- `evidence_basis`
- `rationale_confidence`
- `TODO_VERIFY`
- `reading_action`

## Evidence Policy

- Abstract-supported records receive problem/method/contribution summaries.
- Conclusion-supported records can use conclusion evidence when available.
- Title-only or metadata-only records stay caveated and do not receive invented method or contribution claims.

## Noninterference

No ranking, score, threshold, taxonomy, source fetcher, or workflow trigger semantics are changed.
