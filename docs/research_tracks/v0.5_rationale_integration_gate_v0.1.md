# v0.5 Rationale Integration Gate v0.1

Status: gate definition.

Integration may proceed only when:

- helper tests pass;
- Python 3.11 compatibility is verified;
- Daily JSON compatibility is preserved;
- Daily Markdown ordering is unchanged;
- Weekly synthesis remains unchanged unless explicitly patched later;
- monthly synthesis remains design-only unless implemented in a separate controlled phase;
- no source fetcher, source-health, ranking, taxonomy, query expansion, or negative-keyword semantics change;
- no manual annotation workflow is introduced;
- no external LLM or Supervisor-Skills runtime dependency is introduced.

Decision for Phase 13I:

`eligible_for_future_controlled_rationale_patch`.

This does not mean production is fully ready. It means the helper and proposal are sufficiently bounded for a future controlled implementation patch.
