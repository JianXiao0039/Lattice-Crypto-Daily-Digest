# v0.4 Planning Scope v0.1

## Goal

Improve daily/weekly reliability reporting and research-track handoff quality without changing ranking semantics or creating background automation.

## In Scope

- Source-health reliability.
- Source-starved false-success prevention.
- IACR latest recovery diagnostics.
- Semantic Scholar enrichment diagnostics.
- Daily reliability dashboard.
- Weekly handoff quality.
- Module-SIS chameleon hash track usefulness.
- Manual-only recovery commands.
- Release hygiene.

## Out of Scope

- Private PhD application tooling.
- Advisor emails or SoP drafts.
- Automatic background services.
- Source ranking overhaul unless required by a reliability bug.
- Default single-paper deep-reading engine.
