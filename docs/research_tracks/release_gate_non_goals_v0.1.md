# Release Gate Non-Goals v0.1

Phase 13P does not:

- create, move, delete, or recreate tags;
- run Git write operations;
- modify source fetchers;
- modify ranking scores or thresholds;
- modify taxonomy semantics;
- modify query expansion;
- modify negative keyword behavior;
- create manual annotation workflows;
- create human-gold metric workflows;
- develop shadow classifier productionization;
- create scheduled tasks, cron jobs, watchers, startup tasks, background services, or automatic future runs;
- vendor helper frameworks;
- add external LLM runtime calls;
- touch private PhD, ResearchArtifacts, or ResearchOS paths.
